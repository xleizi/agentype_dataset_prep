# Agentype Benchmark Dataset Preparation

Scripts to reproduce the benchmark dataset used in the Agentype paper:
***"Agentype: A multi-agent LLM framework for single-cell RNA-seq cell type annotation"***

## Overview

These scripts build a harmonized single-cell benchmark from four public databases:

| Database | Species | Raw cells | Final files | Reference |
|----------|---------|-----------|-------------|-----------|
| DISCO | Human | 7.4 M | 28 (.rds) | [Li et al. NAR (2022)](https://doi.org/10.1093/nar/gkab1008) |
| HCL | Human | 599,926 | 30 (.rds) | [Han et al. Nature (2020)](https://doi.org/10.1038/s41586-020-2157-4) |
| MCA | Mouse | 333,778 | 23 (.rds) | [Han et al. Cell (2018)](https://doi.org/10.1016/j.cell.2018.02.052) |
| **Tabula Sapiens** | Human | 483,152 | 25 (.rds) | [Consortium. Science (2022)](https://doi.org/10.1126/science.abl4896) |

**Final output**: 106 Seurat RDS files across 44 tissues, with cell types having ≥100 cells retained.

## Data download

Due to file size constraints, the raw data must be downloaded from their original sources before running these scripts:

- **DISCO**: <https://www.immunesinglecell.org/>
- **HCL**: <http://bis.zju.edu.cn/HCL/>
- **MCA**: <http://bis.zju.edu.cn/MCA/>
- **Tabula Sapiens**: <https://tabula-sapiens-portal.ds.czbiohub.org/>

Place the downloaded files into the corresponding subdirectories (`01_DISCO/`, `02_HCL/`, `03_MCA/`, `06_TabulaSapiens/`) as described in the directory structure below.

The processed RDS benchmark files are not redistributed in this repository because of their large file size and because all source datasets are publicly available. Instead, this repository provides the exact scripts and instructions required to regenerate the processed benchmark RDS files from the original public datasets.

## Running the pipeline

Execute scripts in order:

```bash
# Step 0: Merge metadata into HCL and MCA h5ad files
python3 scripts/00_process_HCL_MCA_metadata.py

# Step 1: Split HCL by tissue
python3 scripts/01_split_HCL_by_tissue.py

# Step 2: Split MCA by tissue
python3 scripts/02_split_MCA_by_tissue.py

# Step 3: Process Tabula Sapiens (build Seurat, split by organ)
Rscript scripts/03_process_TabulaSapiens.R

# Step 4: Convert h5ad to Seurat RDS
Rscript scripts/04_convert_h5ad_to_seurat.R

# Step 5: Organize into benchmark directory tree
Rscript scripts/05_organize_benchmark.R

# Step 6: Filter low-abundance cell types (< 100 cells)
python3 scripts/06_filter_celltypes.py
```

**Expected runtime**: ~2–3 hours on a machine with 64 GB RAM and 12 CPU cores.
Most time is spent on Step 3 (Tabula Sapiens TSV → Seurat) and Step 4 (h5ad → RDS conversion).

## Filtering criteria

Cell types with fewer than **100 cells** in a given tissue file are removed
(minimum threshold for FindAllMarkers in the downstream Agentype pipeline).
After filtering, 6 files containing only a single remaining cell type are excluded.
Final output: 106 RDS files, 7,424,264 total cells (99.79% cell retention).
